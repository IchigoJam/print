Répondre avec des Chiffres / INPUT

Un programme qui résout divers problèmes d'addition en utilisant "RND" (aléatoire).
Utilisez "LET" pour stocker des nombres de A à Z.
Exécutons « RUN » plusieurs fois !
Pour entrer les chiffres à l'aide du clavier, utilisez la commande "INPUT".
Devant le "?", tapez un nombre avec le clavier et appuyez sur la touche ENTREE.
Le nombre entré pour C est mémorisé !
Pour vérifier les réponses, "IF" est utilisé pour juger.
Utilisons "ELSE" lorsque vous voulez montrer un échec.
Il peut également apparaître avec un texte de question.
Essayez-le!
1. Modifions-le pour faire l'addition de 2 chiffres.
2. Modifions-le pour résoudre un problème extrêmement facile
3. Modifions-le pour résoudre un problème de multiplication
4. Mesurons le temps de réponse en utilisant "TICK"
5. Faisons un jeu de frappe de nombre
