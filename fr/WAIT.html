Contrôler le temps / WAIT

Apprenons "wait" sur IchigoJam.
Un curseur carré clignotant à l'écran signifie que le clavier est prêt être utilisé.
Avez-vous vu que ce curseur a disparu depuis peu de temps ? Essayons encore.
Si vous modifiez le nombre après "WAIT", la durée de "wait" pourra changer.
※1 second = 60
Le curseur a disparu plus longtemps qu'avant.
Allumez la LED et éteignez automatiquement.
Utilisez les deux points ' : ' (facile à confondre avec le point-virgule ' ; ') pour concaténer(joindre) les commandes.
Allumez avec "LED1", attendez avec "WAIT120", éteignez avec "LED0".
Pour allumer pour peu de temps LED.
Si vous modifiez le nombre pour "WAIT", cela modifie le temps d'allumage de la LED.
Allumez la LED deux fois..
Vous pouvez vous joindre plus de commandes avec les deux points " : ".
Essayez-le!
1. Faisons "WAIT" pour 10 secondes
2. Allumez la LED pendant 5 secondes et éteignez-la
3. Faisons "WAIT 10000", au milieu, appuyez sur la touche ESCAPE pour arrêter
4. Allumez pour longtemps et allumez pour peu de temps
5. Essayons de créer un motif lumineux LED attrayant
