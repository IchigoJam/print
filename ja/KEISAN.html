まとめてでんたく　INPUT

まとめてでんたくを作って計算をしよう INPUTとPRINTを使うことでできるよ
10 INPUT A 20 INPUT B
10 INPUT Aで一つ目の代数を入力するよ
20 INPUT Bで二つ目の代数を入力するよ
次で四則計算をするよ
30 C=A+B 40 D=A-B 50 E=A*B 60 F=A/B 70 G=A%B
割り算の計算ではあまりが出ないのであまりの計算もしよう
計算けっかを出力しよう
80 PRINT C 90 PRINT D 100 PRINT E 110 PRINT F 120 PRINT G
RUNで実行してみよう
やってみよう！
