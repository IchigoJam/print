えいごのあいさつをマスターしよう IF

INPUT（インプット）コマンドをつかって、 じかん（タイム）をにゅうりょくできるようにしよう！
10 INPUT A 20 PRINT "TIME:";A
いまのTIMEをいれてみようか。
ごぜん・ごごをえいごでおしらせしてくれるようにしよう。
30 IF A<12 PRINT"AM" 40 IF A>=13 ?"PM"
ごぜんはAM、ごごはPM。 にほんは、0~24じ（ごぜん0~12じ、ごご13~24じ）ということがあるけど、
えいごでは、AM0~12じ、PM0~12じとひょうきするんだ。ややこしくてごめんね。
もし、21じ＝PM9じになったらこうひょうじしよう。　
50 IF A=21 ?"MOU9JI!"
おきていると、おとなにおこられるよ！もうねよう。
かずを、にほんごのあいさつにあてはめよう！
60 INPUT "0:OHAYOU 1:KONNICHIHA 2:KONBANHA 3:OYASUMI ",S
いまのじかんにあった、あいさつはえらべた？
もし、AM0~5じにゲームをしていたとしたら、どうなるかな？
70 IF A>=0 AND A<=5 ?"HAYAKUNENASAI!"
おこられた！ねます！
もうひとくふう！えいごであいさつをかこう！
80 IF A>=6 AND A<=11 AND S=0 ?"GOOD MORNING!" 90 IF A>=12 AND A<=17 AND S=1 ?"GOOD AFTERNOON！🍙" 100 IF A>=18 AND A<=20 AND S=2 ?"GOOD EVENING:)" 110 IF A>=21 AND A<=23 AND S=3 ?"GOOD NIGHT!🚀"
