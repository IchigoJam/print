# かずあてゲーム

## [ランダム](RND.html)なかず、0から9をあてるゲーム

```ij
10 A=RND(10)
20 ?A
```

## カンであてるゲームにしよう

```ij
20 INPUT N
30 IF A=N ?"BINGO!":END
40 GOTO 20
```

## スコアをつけよう

```ij
15 S=100
30 IF A=N ?"BINGO! S=";S:END
39 S=S/2
```
まちがえるごとにスコアははんぶん！

## ヒントがほしい？

```ij
31 IF N>A ?"TOO BIG"
32 IF N<A ?"TOO SMALL"
```
- いれたかず(N)がこたえ(A)よりおおきいときは、おおきすぎ(too big)
- いれたかず(N)がこたえ(A)よりちいさいときは、ちいさすぎ(too small)

## わかりやすく

```ij
20 INPUT "KAZUATE 0-99?",N
```

## チャレンジ

1. 0-99まであてるゲームにしてみよう
2. ヒントをふやしてみよう
3. まちがえたかいすうをひょうじしてみよう
