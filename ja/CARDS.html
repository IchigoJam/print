なんのカード？

あいてがなんのトランプをひいたかあててみよう！まずはマーク！
1から4のかずをランダムにせってい。
わかりやすいように、それぞれのかずにマークをつけてあげよう。
10 A=RND(4)+1 20 ?"SELECT ♥=1 ♦=2 ♣=3 ♠=4"
どんなマーク？
あたったら、すうじをあてるぎょうにとぶ。
はずれたら、ざんねん！さいチャレンジ！
30 INPUT "MARK?",B 40 IF B=A GOTO80 50 IF B!=A ?"WRONG!AGAIN!" 60 WAIT120 70 GOTO10
つぎはすうじ！マークとおなじように、1から13のかずをせってい。
80 C=RND(13)+1
どんなすうじ？ あたったらクリア！はずれたら、おしい！もういちど！
90 INPUT "NUMBER?",D 100 IF D=C ?"CLEAR!" 110 IF D!=C ?"CLOSE!AGAIN!"
さいしょにもどろう。
120 WAIT120 130 GOTO10
やってみよう！

2. マークとすうじにヒントをつけてみよう
3. ヒントをひょうじするかしないかえらべるようにしてみよう
4. 【むずかしい】クリアしたときにあてたカードをひょうじしてみよう
