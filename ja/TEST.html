なんのカード？

あいてがなんのトランプをひいたかあててみよう！
まずはどのマークかをあてよう！
1から4のかずをランダムにせってい。
わかりやすいように1から4のかずにマークをつけてあげよう。
10 A=RND(4)+1 20 ?"SELECT ♥=1 ♦=2 ♣=3 ♠=4"
どんなマーク？
30 INPUT "MARK?",B
あたったら、すうじをあてるぎょうにとぶ。
はずれたら、ざんねん！もういちど！
さいしょのぎょうにもどって、すぐにさいチャレンジできるようにする。
40 IF B=A GOTO80 50 IF B!=A ?"WRONG!AGAIN!" 60 WAIT120 70 GOTO10
すうじをあてよう！
マークとおなじように、1から13のかずをせってい。
80 C=RND(13)+1
どんなすうじ？
90 INPUT "NUMBER?",D
あたったらクリア！おめでとう！
はずれたら、おしい！もういちど！
100 IF D=C ?"CLEAR!" 110 IF D!=C ?"CLOSE!AGAIN!"
さいしょにもどろう。
120 WAIT120 130 GOTO10
なかなかあたらない！じゃあヒントをつけてみよう。
マークがあか(つまり1か2)なら"RED"、くろ(つまり3か4)なら"BLACK"とひょうじ。
25 IF A<=2 ?"HINT:RED" 26 IF A>=3 ?"HINT:BLACK"
やってみよう！
1. 15ぎょうめにCLSといれるとどうなるかな？
2. すうじにもヒントをつけてみよう
3. ヒントをひょうじするかしないかえらべるようにしてみよう
4. 【むずかしい】クリアしたときにあてたカードをひょうじしてみよう
5. 【むずかしい】さいチャレンジをしゅどうでおこなえるようにしてみよう
